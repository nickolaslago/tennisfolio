# Contributing to Tennisfolio

Tennisfolio is an open-source, local-first tennis performance tracker.

## Setup
See the README for local dev setup (pnpm workspace: apps/web, apps/api, packages/core).

## Workflow
1. Create a branch off `main`.
2. Make your changes, commit with a clear message.
3. Open a pull request.

## Code style
- Web: TypeScript, React, Tailwind.
- API: Python, FastAPI, type-hinted.

## Reporting issues
Open a GitHub issue with steps to reproduce.
